# 校园外卖系统 - UniTakeout

一个现代化的校园外卖点餐平台，支持点餐、委托广场和AI智能推荐功能。

## 功能特性

### 🍔 基础点餐功能
- 店铺列表浏览
- 商品详情查看
- 购物车管理
- 订单创建和管理

### 📋 委托广场
- 发布求带委托：用户可以发布需要他人帮忙带餐的需求
- 发布捎带能力：学生可以声明自己能帮忙带什么店铺的东西
- 委托匹配和接受
- 促进校园餐饮消费和社交互动

### 🤖 AI智能推荐
- 智能选品推荐：根据用户需求自动推荐合适的美食
- 自动点单：AI根据用户偏好自动生成订单
- 对话式交互：通过自然语言与AI交互

### 👤 用户中心
- 用户登录注册
- 个人信息管理
- 订单历史查看
- 收货地址管理

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **HTTP客户端**: Axios
- **样式**: CSS3 (现代简洁风格)

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发环境运行

```bash
npm run dev
```

### 生产环境构建

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 环境变量

创建 `.env` 文件配置API基础URL：

```env
VITE_API_BASE_URL=http://localhost:3000/api
```

## 主要页面

- **首页** (`/`): 展示热门店铺和快捷入口
- **店铺列表** (`/shops`): 浏览所有店铺
- **店铺详情** (`/shop/:id`): 查看店铺商品并点餐
- **订单管理** (`/order`): 查看和管理订单
- **委托广场** (`/delegation`): 发布和浏览委托信息
- **AI助手** (`/ai-agent`): AI智能推荐和自动点单
- **个人中心** (`/profile`): 用户信息和设置

## UI设计特点

- **现代简洁**: 采用现代化的设计语言，界面简洁清爽
- **响应式设计**: 适配移动端和桌面端
- **交互友好**: 流畅的动画和过渡效果
- **色彩搭配**: 主色调为绿色(#07c160)，符合外卖行业特点
